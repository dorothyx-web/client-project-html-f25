Interview Prep for UX / Interaction Design

Behavioral: Use the STAR Method

Situation (context) → Task (goal) → Action (what you did) → Result (impact + metric). Keep answers to ~90 seconds.

  • Design trade-off prompt: “Tell me about a time you had conflicting user and stakeholder needs.”
  • Teamwork prompt: “Describe a conflict on a project and how you resolved it.”

Portfolio Walkthrough (3–5 minutes)

Suggested Outline

  1. One-sentence problem + who the users are
  2. Your role + collaborators + timeline
  3. Process highlight: research → insights → decisions → tests
  4. Outcome & impact: what changed, include numbers if possible

Whiteboard / Critique Exercises

Think aloud, scope the problem, state assumptions, and aim for a workable flow, not pixel perfection.

  • Ask clarifying questions, propose success metrics.
  • Sketch primary flow + edge cases; label decisions.
  • Invite feedback and iterate visibly.

Smart Questions to Ask Interviewers

  • “How do research insights flow into the roadmap?”
  • “What does a successful first 90 days look like in this role?”
  • “How are design decisions documented and shared across teams?”
